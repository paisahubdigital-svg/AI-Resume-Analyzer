# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer built using Python and Google Colab.

## ✨ Features

- 📄 Upload Resume PDF
- 🔍 Extract Skills
- 📊 ATS Score
- ❌ Missing Skills Detection
- 📈 Resume Analysis Chart
- 💡 Resume Improvement Suggestions

## 🛠️ Tech Stack

- Python
- Google Colab
- PyMuPDF
- spaCy
- Pandas
- Matplotlib

## 🚀 How to Run

1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Run all cells.
4. Upload a PDF resume.
5. View the ATS score and skill analysis.
