# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer built using Python and Google Colab.

## ✨ Features

- 📄 Upload Resume PDF
- 🔍 Extract Skills
- 📊 ATS Score
- ❌ Missing Skills Detection
- 📈 Resume Analysis Chart
- 💡 Resume Improvement Suggestions

## 🛠️ Tech Stack

- Python
- Google Colab
- PyMuPDF
- spaCy
- Pandas
- Matplotlib
